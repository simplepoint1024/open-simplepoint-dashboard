Set up Two-Factor Authentication

  1. Open Authy or Google Authenticator on your phone.
  2. Add a new account and choose to enter a setup key or scan a QR code.
  3. Use the secret below or generate a QR code from the otpauth URL.

SECRET
otpauth URL:
otpauth://totp/...

You can use any QR code generator tool to turn the otpauth URL into a QR code for scanning.

After adding the account in your authenticator app, log out and sign in again.
If 2FA is enabled correctly, you will be prompted on the next login to enter the 6-digit code on the /two-factor-verify page.
